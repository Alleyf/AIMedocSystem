AIMeDocSys

AIMeDocSys是基于ElasticSearch分布式搜索分析引擎和光学字符识别技术(OCR)开发的医学文献智能识别检索系统，解决医疗行业中医学文献检索效率低、人力物力消耗大的问题。系统可建立本地文献库，支持批量上传、删除、检索本地文献并在线查看、下载，同时支持关键词查询知网和万方文献。采用光学字符识别技术可准确提取文献中图片所含内容，检索内容精确到页数，并提供段落摘要。大数据可视化技术展示系统运行状态、文献分类、点击量等数据，基于生成型预训练变换模型实现文献自动分类和智能问答助手服务，使用Neo4j图形数据库构建医学知识图谱为用 户提供疾病衍生的各种关系。系统主体在Windows操作系统上开发，通过Docker容器化部署至腾讯云服务器，实现快速自动化和集群部署.
本系统可为医生、医学生、医药代表、医疗团队等提供全面优质的文献管理和医学信息检索服务

Technology

九大主流技术，实现整个系统的设计到部署.

{# #}

ElasticSearch搜索引擎

AIMeDocSys主要是利用了ElasticSearch分布式搜索引擎对用户上传的文献的文字文本和图片文本分词并建立倒排索引，从而实现了在搜索过程中达到高性能、速度快、低延迟的要求.

Read More

光学字符识别技术(OCR)

AIMeDocSys基于百度飞桨的预训练模型，通过自制医学文献图片数据集和开源公开数据集对模型进行微调，进一步提升PaddleOCR模型检测与识别模型的精度，经过290张以上医学文献图片准确率的测试达到94.93%以上.

Read More

Neo4j高性能图形数据库

AIMeDocSys采用基于Neo4j高性能的NoSQL图形数据库并且使用刘焕勇老师整理的医学数据文件搭建便捷可查的可视化的医学知识图谱，帮助用户简单方便的获取关键医学信息.

Read More

Redis持久化缓存机制

AIMeDocSys采用Redis对网站的静态资源和用户搜索后的前端数据进行持久化缓存，该机制能够显著提升网页的响应速度，提升用户体验.

Read More

Docker容器化部署

AIMeDocSys在Windows操作系统上开发调试并采用Docker容器化打包部署至腾讯云服务器上，实现了方便快速并且与平台解耦的自动化部署方式.

Read More

Nginx网关服务分发

AIMeDocSys通过加入Nginx网关服务分发机制和必要逻辑与中间件如消息队列等，能够处理好限流、负载均衡、过载保护、熔断、降级、服务解耦等挑战实现高并发、高可用.

Read More

Vue渐进式JavaScript框架

AIMeDocSys采用Vue渐进式JavaScript框架打造后台管理系统，以友好的人机交互响应和精美的前端设计辅助管理人员对系统的管理更加省心.

Read More

Django开源Web应用框架

AIMeDocSys使用Django后端框架独立开发与前端解耦，实现前后端分离，利用其丰富的中间件进行鉴权、消息队列、负载均衡、高可用性监视等.

Read More

Echarts开源可视化图表库

AIMeDocSys使用Echarts构建数据可视化大屏界面，直观生动地展现本系统中的各种数据信息，为用户提供重要的信息，便捷用户管理，提升用户体验.

Read More

关于
